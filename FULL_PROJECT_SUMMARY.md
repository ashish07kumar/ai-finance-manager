## 🚀 Quick Start

### Terminal 1 - Backend
```bash
cd finance-tracker-backend
npm install
npm start
```
**Runs on**: http://localhost:5000

### Terminal 2 - Frontend
```bash
cd finance-tracker-frontend
npm install
npm run dev
```
**Runs on**: http://localhost:3000
